# Checklist UX avant release ✅

## Navigation

- CTA principal visible sans scroll.
- Etats vides couvrent matieres, classes, boutique.
- Retour arriere restaure l'etat.

## Performance

- LCP < 2.5s sur mobile 3G simulee.
- Pas de flash de contenu non style (FOUC).
- Images optimisees et lazy chargees.

## Accessibilite

- Focus visible partout.
- Contrastes AA valides.
- `prefers-reduced-motion` respecte.

## Feedback

- Toasts pour succes/erreur critiques.
- Recap clair en fin de quiz.

## Copy

- Ton chaleureux, pas de jargon.
- Emojis limites mais presents pour chaleur.

## Tests manuels

- Onboarding complet en 2 min.
- Survie: timer et coeurs cohérents.
- Classe: podium et partage fonctionnels.
