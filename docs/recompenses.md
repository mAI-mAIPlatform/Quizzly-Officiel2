# Systemes de recompenses 🎁

## Types

- XP: progression lineaire, utilisee pour debloquer badges.
- Badges: jalons visibles, partageables.
- Monnaie douce: pieces pour cosmetiques boutique.
- Pass saison: paliers limites dans le temps.

## Principes

- Recompense toujours rattachee a un effort clair (quiz termine, serie, defis).
- Affichage au moment de l'effort, pas uniquement dans un recap lointain.
- Eviter l'inflation: ne pas multiplier les monnaies.

## Courbes

- XP: +10 par bonne reponse, +5 pour serie de 3, +20 bonus de fin.
- Badges: seuils publics (5 quiz termines, 3 jours de suite, 1ere place classe).
- Monnaie douce: petites quantites quotidiennes pour retention.

## Anti frustration

- Pas de perte de XP en cas d'echec; proposer une astuce.
- Limiter le grind: cap journalier doux mais afficher le cap.

## UI

- Progress bars courtes avec jalons visibles.
- Boutons "equiper" pour cosmetiques immediatement apres achat.
