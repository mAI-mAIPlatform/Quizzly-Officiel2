# Principes UX 😊

## Ton

- Chaleureux, direct, sans jargon.
- Encourager plutot que sanctionner; reformuler les erreurs en opportunites.

## Rituel d'ecran

- 1 action primaire, 1 secondaire.
- Texte <= 12 mots par ligne, police lisible.
- Feedback immediat pour chaque action (couleur, vibration, son leger).

## Cohesion visuelle

- Couleurs: palette contrastes AA.
- Iconographie simple, toujours labelisee.
- Boutons larges sur mobile, espacement >= 12 px.

## Navigation

- Sidebar pour desktop, barre basse pour mobile.
- Etat actif clair; jamais plus de 2 niveaux d'imbrication visibles.

## Contenu

- Toujours afficher la valeur concrete: points gagnes, temps estime, badges.
- Eviter le vide: placeholders utiles, suggestions de quiz.

## Performance percue

- Prefetch des quiz suivants apres la question 2.
- Skeletons au lieu de spinners pour les listes longues.

## Checklist rapide

- Action principale visible sans scroll.
- Boutons desactivables mais explicites ("plus de vies").
- Erreurs en langage naturel + action de resolution.
