# Boucles de feedback 🔁

## Pourquoi

Des boucles courtes maintiennent la motivation et expliquent la progression.

## Boucles clefs

- Reponse immediate: validation + explication courte.
- Fin de quiz: recap XP, badge en vue, suggestion suivante.
- Hebdomadaire: rapport de progression avec streaks.
- Sociale: notification quand un ami bat votre score.

## Delais cibles

- Feedback visuel < 150 ms.
- Recap complet < 500 ms.
- Notification push envoye <= 60 s apres evenement.

## Composants associes

- `ToastSuccess` pour micro feedback.
- `ModalRecap` pour fin de quiz.
- `BannerStreak` pour series.

## Copy exemples

- Bonne reponse: "Bien joue ! +10 XP"
- Mauvaise: "Presque. Essaie cette astuce: ..."
- Survie: "Nouveau record, partage-le !"
