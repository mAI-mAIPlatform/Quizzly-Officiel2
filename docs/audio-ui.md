# Audio UI 🔊

## Roles du son

- Confirmer une action.
- Marquer la tension (timer bas).
- Celebrer une victoire.

## Palette sonore

- Click leger pour navigation.
- Ding doux pour bonne reponse.
- Basse courte pour erreur.
- Fanfares courtes pour badge.

## Volumes

- 60% par defaut, reglage dans Reglages.
- Mute rapide dans Navbar.

## Performance

- Precharger les sons critiques (bonne/mauvaise reponse).
- Utiliser formats courts (.mp3/.ogg) < 50 Ko.

## Accessibilite

- Toujours offrir feedback visuel alternatif.
- Sous-titres pour videos.
