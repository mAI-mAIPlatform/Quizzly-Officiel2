# Strategie de retention 📆

## Leviers principaux

- Streaks quotidiennes avec rappel doux.
- Defis hebdo par matiere.
- Contenu frais: 3 nouveaux quiz par semaine.
- Social: comparer ses badges avec amis/classe.

## Cadence suggeree

- Jour 1: onboarding + badge accueil.
- Jour 2: notif streak + defis faciles.
- Semaine 1: defi classe + nouveau badge maitrise.
- Semaine 2: saison Pass nouveau palier.

## UX

- Mettre en avant "reprendre" sur dashboard.
- Bannieres dynamiques pour defis actifs.
- Timer clair sur elements limites dans le temps.

## Mesure

- Retention D1/D7/D14.
- % joueurs ayant 2 badges+.
- Participation aux defis hebdo.
