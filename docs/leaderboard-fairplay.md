# Classements et fair-play 🏆

## Types de leaderboards

- Classe: scores cumules par quiz classe.
- Amis: meilleurs scores hebdo.
- Global: top 100 par mode Survie.

## Principes d'equite

- Anti triche: rafraichir le score depuis serveur; detecter latences suspectes.
- Anti inflation: reset hebdo pour amis, reset saison pour global.
- Mode debutant: cacher le global jusqu'a 3 quiz finis.

## UX

- Filtre clair (Classe / Amis / Global).
- Indiquer le temps restant avant reset.
- Highlight de la position du joueur meme hors top 10.

## Feedback

- Micro-copy: "Tu es a 150 pts du top 10".
- Bouton "Rejouer ce quiz" depuis la ligne du tableau.

## ModERATION

- Pseudonymes filtres (liste noire), avatars securises.
- Bouton signaler.
