# Notifications 🔔

## Types

- Rappel de streak: "Plus que 2h pour garder ta serie".
- Progression: "Tu as debloque le badge Intermediaire".
- Social: "Alex a battu ton score en Survie".
- Classe: "Le quiz de la semaine est disponible".

## Cadence

- Max 1 push par jour par type.
- Heure recommandee: 18h locale (retour maison).

## Contenu

- Message court (<60 caracteres), CTA clair.
- Profiter des emojis legers pour chaleur.

## Opt-in

- Demande apres premiere victoire; expliquer la valeur.
- Toujours un toggle dans Reglages.

## Anti spam

- Fermer la boucle: ne pas relancer si action deja realisee.
