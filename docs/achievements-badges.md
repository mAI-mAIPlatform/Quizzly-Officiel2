# Badges et achievements 🏅

## Objectif

Fournir des jalons clairs qui celebrent la progression et incitent a continuer.

## Categories

- Progression: terminer X quiz, finir un chapitre.
- Maîtrise: 90%+ sur 3 quiz consecutifs.
- Social: inviter un ami, rejoindre une classe.
- Endurance: streak de 7 jours, 3 runs Survie.

## Design

- Nom + icone + condition simple.
- Couleur par rarete (commun, rare, epique).
- Tooltip expliquant comment le gagner.

## Affichage

- Grille sur page Profil.
- Banner pop quand badge obtenu avec bouton "partager".
- Tag visible dans Classement.

## A/B

- Tester nombre de badges visibles au lancement (6 vs 12).
- Tester message de pop (confetti vs vibration).

## Maintenance

- Ajouter max 1 nouveau badge par sprint pour eviter l'inflation.
- Deprecier proprement: badge grise + mention "legacy".
